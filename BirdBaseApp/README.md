# BirdBase App (Edge Node)

The mobile client for the BirdBase platform, built with **Flutter** and **Dart**.

## Features
* **Zero-Latency Offline Execution:** Utilizes a lightweight `YOLOv8n` TFLite model directly on the device's processor.
* **Offline Database:** A built-in JSON mapping system instantly retrieves detailed information from over 2,600 bird species without requiring an internet connection.
* **Clean UI/UX:** Designed for quick, on-the-go field usage with one-tap analysis.
